# Backend Fundamentals & Architectural Choices

This project is a backend-focused assignment that demonstrates understanding of core backend concepts, architectural patterns, and decision-making in real-world scenarios.

## Objective
The goal of this assignment is to analyze and compare different backend technologies and approaches, and to apply them in practical, scenario-based situations.

---

## Topics Covered

- REST vs GraphQL APIs  
- Database Design (SQL vs NoSQL)  
- Authentication using JWT  
- Microservices Architecture  
- Caching Strategies (Client-side, Server-side, CDN)  
- API Design & Performance Optimization
